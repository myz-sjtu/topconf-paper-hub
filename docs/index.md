## Snapshot

This page is populated by the static export pipeline.

```bash
./scripts/build_static_site.sh
```

The generated site contains:

- Paper metadata grouped by conference, year, and domain
- Abstracts when available
- Source links for DBLP, OpenReview, CVF, ACM DL, USENIX, and OpenAlex
- Static JSON data under `/data/papers.json`
